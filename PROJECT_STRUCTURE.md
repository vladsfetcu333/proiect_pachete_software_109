# Structura proiectului

Acest proiect contine doua parti:

- aplicatia Python Streamlit pentru analiza unui magazin online;
- scriptul SAS pentru analiza aceluiasi set de date si generarea de rapoarte.

Setul de date inclus este `data/online_retail_project.csv`, un set sintetic realist construit pe modelul tranzactiilor unui magazin online european. Datele contin comenzi, clienti, tari, produse, cantitati, preturi, discounturi si costuri de transport.
